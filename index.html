📋 Google Cloud Setup Guide

  1. Go to console.cloud.google.com and sign in
  2. Click Select a project → New Project → name it anything → Create
  3. In the left menu go to APIs & Services → Library
  4. Search Google Drive API → click it → Enable
  5. Search Google Docs API → click it → Enable
  6. Go to APIs & Services → OAuth consent screen
  7. Choose External → fill in App name (anything) and your email → Save
  8. Go to APIs & Services → Credentials
  9. Click + Create Credentials → OAuth client ID
  10. Application type: Web application
  11. Under Authorized JavaScript origins add: this page's URL
  12. Click Create → copy the Client ID
  13. Paste the Client ID in the field above and click Save
